BACKEND JAVASCRIPT DIRECTORY

PURPOSE:
This directory contains all Firebase and backend JavaScript files.

FILES:

1. firebase-config.js
   - Firebase project initialization
   - SDK setup and configuration
   - Database and authentication service setup
   - Export Firebase services for use in other files
   - DO NOT commit API keys (use .env file)

2. auth.js
   - User authentication logic
   - Sign up functionality
   - Login/logout flows
   - Password reset
   - Session management
   - User profile management
   - Permission/role checking

ADDITIONAL FILES TO CREATE:
- database.js: Database operations (CRUD)
- storage.js: File upload and download logic
- validators.js: Data validation functions
- error-handler.js: Centralized error handling
